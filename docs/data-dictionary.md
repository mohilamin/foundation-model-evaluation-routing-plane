# Data Dictionary

Important generated datasets:

- `evaluation_tasks.csv`: synthetic tasks, domains, constraints, golden labels, and slices.
- `routing_demo_requests.csv`: subset used for routing simulations.
- `model_registry.csv`: model candidate metadata.
- `simulated_model_outputs.csv`: model/task simulated outputs and metrics.
- `model_evaluation_report.csv`: model-level scorecard.
- `model_slice_report.csv`: model performance by task/domain/risk slice.
- `routing_decisions.csv`: selected model, fallback, rejected models, utility components, and explanations.
- `pareto_frontier_report.csv`: model tradeoff analysis.
- `canary_evaluation_report.csv`: canary versus baseline comparison.
- `model_monitoring_report.csv`: simulated model health and routing volume.
