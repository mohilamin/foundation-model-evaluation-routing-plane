# Three-Minute Demo Script

## 0:00-0:30 Business Problem

Explain that production AI teams need model routing, not one fixed model for every task.

## 0:30-1:00 Model Registry

Show model candidates with different cost, latency, safety, and domain profiles.

## 1:00-1:45 Routing Lab

Enter a high-risk healthcare or finance request. Show filtering, selected model, fallback, rejected models, and explanation.

## 1:45-2:20 Pareto Frontier And Canary

Show quality/cost tradeoffs and the canary rollout recommendation.

## 2:20-3:00 Deployment Readiness And Scorecards

Show scorecards, tests, Docker, CI, and Streamlit/FastAPI readiness.
